Gradle 2 (Groovy 2), Wildfly 8 or 9, Java 7 or 8, TDD with JUnit, Spock, PMD, JaCoCo, FindBug, JDepend, CodeNarc and Checkstyle, Eclipse and IntelliJ Idea support in Gradle activated

Wicketstuff 7, Wicket Bootstrap 3, Wicket CDI 1.1, JQuery Mobile 1.4

Java EE 7

  • JPA 2 (Hibernate and EclipseLink in config)
  • JSF 2 (with Contracts and Versions)
  • Primefaces 5 (with Extensions)
